# MCP3201 时序参数（3.3V 供电）

> 提取自数据手册 `C49274_模数转换芯片ADC_MCP3201-BI-SN_规格书_WJ350465.PDF`
> 供电电压：VDD = 3.3V

## 1. 极限参数（3.3V 推算）

数据手册仅明确给出 5V 与 2.7V 两档的 fCLK(max)：

| 条件 | fCLK(max) |
|------|-----------|
| VDD = 5V | 1.6 MHz |
| VDD = 2.7V | 0.8 MHz |

**3.3V 保守取值：**
- 线性插值估算：≈ 1.0 MHz
- 本设计取 **SPI 时钟 = 1.0 MHz**（50 MHz 系统时钟 50 分频），留有充足裕量。

| 参数 | 符号 | 最小值 | 典型值 | 最大值 | 单位 | 3.3V/50MHz 设计值 |

## 3. 转换时序（FIGURE 5-1）

```
CS    ─┐                                              ┌─
       └──────────────────────────────────────────────┘
CLK       ┌──┐  ┌──┐  ┌──┐  ┌──┐  ┌──┐  ┌──┐  ┌──┐
       ───┘  └──┘  └──┘  └──┘  └──┘  └──┘  └──┘  └──
DOUT   HI-Z  NULL  B11  B10  B9   ...  B1   B0   HI-Z
              BIT        ←—— MSB 先输出 ——→
```

| 阶段 | 说明 | 时钟周期数 |
|------|------|------------|
| 采样期 tSAMPLE | CS 下降后，第 1 个 CLK 上升沿开始采样 | 1.5 个 CLK |
| 转换期 tCONV | 逐次逼近（SAR）转换 | 12 个 CLK |
| 数据输出 tDATA | 输出 12 位结果 + 可选 LSB-first 重复 | 12+ 个 CLK |

> 最小完整转换周期：**14 个 SPI 时钟**（1 个采样/NULL + 12 个数据位）。
> 本模块使用 **17 个 SPI 时钟**（850 clk@50MHz = 17 µs），兼容 MSB-first 与 LSB-first 两种读出方式。

## 4. 采样率约束

- 系统时钟：50 MHz
- SPI 时钟：1 MHz（50 分频）
- 单次转换：17 SPI 周期 = 17 µs
- 目标采样率：40 kHz → 周期 25 µs
- **裕量：** 25 µs - 17 µs = **8 µs**（满足 tCSH 最小 625 ns）

## 5. 最小时钟频率警告

数据手册 Note 3：
> 采样电容会随时间泄漏电荷，有效时钟频率低于 **10 kHz** 时会影响线性度，尤其在高温下。

本设计 SPI 时钟 1 MHz 远高于此限制，无需担心。

## 6. SPI 模式兼容性

- 支持 **Mode 0,0**（CLK 空闲低）与 **Mode 1,1**（CLK 空闲高）
- MCP3201 在 **CLK 下降沿**输出数据
- FPGA/MCU 应在 **CLK 上升沿**锁存数据
